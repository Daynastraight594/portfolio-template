## 🖥️ What this app does

portfolio-template is a single-file portfolio app you can use to present your work, skills, and contact details in one place. It is built for people who want a simple personal site without setup work.

It includes:

- Dark and light theme support
- 4-language i18n support
- Scroll animations
- GitHub Pages ready layout
- Zero-dependency structure
- Single-file format for easy use
